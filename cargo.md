
# Options
## Create a new project

```
cargo new <projectname>
```

## Build
```
cargo build             // with debug information
cargo build --release   // release mode
```

## Clean project
```
cargo clean
```

## Run
```
cargo run             // with debug information
cargo run --release   // release mode
```

## Test: run the tests
```
cargo test
```

## Search for a crate
```
cargo search <topic or name>
```
